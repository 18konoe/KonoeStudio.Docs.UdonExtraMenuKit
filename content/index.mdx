# Udon Extra Menu Kit とは

- メニューを出したときに追加の拡張メニューを目の前に出現させる
- ワールド内に持ち運びできる設定画面のようなウィンドウを提供
- それらをベースに、時計や Join ログ、目覚ましなど様々な機能を提供
- 拡張メニューはウィンドウやその他オブジェクトを目の前に呼び出すボタン付き

これらを拡張メニューを中心にひとまとめにしたキットです。
多少の設定は必要となりますが、たくさんのスイッチをひとまとめにしたりできます。
また、まとめた設定ウィンドウを拡張メニューから呼び出すことで、いちいちスイッチを押しにいく必要がなくなります。

設定ウィンドウを作るのに最適な UI コンポーネントを各種用意しており、拡張性が高いのも特徴です。
